HIMS Software Guide

NABH & GCI Compliance Guidelines

  • Batch/Lot Tracking — Every material in must record batch/lot number for traceability (NABH SCM 4.b)
  • Quality Check — All incoming materials must pass quality inspection before stock entry (NABH SCM 6.a)
  • Conditional Storage — Classify storage conditions: Normal, Cool, Cold (2-8°C), Frozen (-20°C), Controlled RT (NABH SCM 8.c)
  • MRN (Material Receipt Note) — Auto-generated MRN for every incoming shipment (GCI Guideline 4.2)
  • Vendor Credentials — Maintain GST, Drug License, ISO certs for all vendors (NABH SCM 3.a)
  • Audit Trail — Every transaction is logged with timestamp and user identity (NABH QM 10.a)
  • Expiry Management — Track expiry dates with visual lifecycle bar; first-expiry-first-out (NABH SCM 9.b)
  • Demand/Requisition Workflow — Employee → HOD → Admin → Storekeeper → Receive (NABH SCM 5.c)

Vendor Code Format

XXX-### e.g. MIH-001 (Company Initials + Sequential Number)

SKU / Barcode Format

DDD-XXX-### e.g. ER-GLV-001 (Department Code + Item Type + Number)

Barcode: 13-digit EAN-13 format starting with 890 (India prefix)

1. Employee raises a demand with item, qty, purpose → 2. HOD reviews and approves/rejects → 3. Admin reviews and approves/rejects → 4. Storekeeper fulfills from stock → 5. Employee confirms receipt → Closed
