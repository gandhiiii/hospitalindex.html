# Hospital Management System

A full-featured hospital management system built with vanilla JavaScript (no frameworks). Data is stored in the browser's localStorage.

**Live demo:** `https://gandhiiii.github.io/Fullstack/`

## Features

- Dashboard with KPIs and widgets
- User Management with roles & permissions
- Department Management
- Feature Rights control
- Inventory with stock management, barcodes, and lifecycle tracking
- Gate Security with visitor/patient/doctor passes, photo capture, HOD approval
- Phase 2 Infrastructure: material tracking, Gantt charts, reports
- Admissions & Room View with color-coded status grid
- Ambulance tracking with Leaflet map
- Complaints & Problems management
- Tasks, Checklists (room + admin)
- Lost & Found
- Projects
- Cross-tab live sync via BroadcastChannel

## Setup

No build tools needed. Open `dashboard.html` in a browser.

Default login: `admin` / `admin123`
